# Full-Stack Note-Taking Application

## Description

This is a note-taking application built using CRUD (Create, Read, Update, Delete) operations. It is designed for efficient note management. The project demonstrates a full-stack workflow, connecting a responsive frontend to a persistent backend storage system.

## Key Features

    * Full-Stack Application: built with Node.js and Express on the backend, paired with HTML, CSS, and JavaScript frontend.
    * Complete CRUD Functionality: users can add, view, edit, and remove notes in real-time.
    * Notes are stored in a data.json file. This ensures that your data remains intact and available even after the server restarts or the browser is closed.
    * The application was deployed on Render.com.
